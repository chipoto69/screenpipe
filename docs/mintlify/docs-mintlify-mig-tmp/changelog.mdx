## week of may 3, 2026

### new features

- **new integrations** — connect [Zoom](/connections), [Supabase](/connections), Jira (now via OAuth 2.0), the [Bee wearable](/connections), and the [Hermes Agent](/connections) (paste your screenpipe MCP straight into Hermes). configure in **settings → connections**.
- **connections promoted to the main sidebar** — manage every integration from a top-level sidebar entry instead of digging through settings.
- **meeting notes, redesigned** — meeting notes is now a top-level sidebar destination with a two-pane notepad, a "coming up" calendar section that auto-enriches the active meeting, a timeline scrubber that replaces the per-row drawer, speaker rename, "open in timeline", and a granola-style single-column layout. [see meeting transcription](/meeting-transcription).
- **embedded agent browser** — the screenpipe-owned browser now lives as a right-side sidebar inside chat, with width and collapsed state persisted per conversation. spoofs Safari UA so sites like x.com load.
- **browser API** — new `POST /navigate` and `POST /snapshot` endpoints on the local browser API for driving the embedded browser from pipes. see the [API docs](/for-developers).
- **clipboard capture privacy toggle** — disable clipboard capture from settings or via the `--disable-clipboard` CLI flag.
- **one-click "delete last N minutes"** — wipe the last 15, 30, or 60 minutes of recordings (audio, video, frames) from the tray.
- **media-only retention mode** — new `--retention-mode` CLI flag keeps search alive while reclaiming disk space. media-only is now the default everywhere. see the [CLI reference](/cli-reference).
- **tray "pause for X minutes"** — pause capture from the tray with a chosen duration, with auto-resume + a live tooltip and a notification when it kicks back in.
- **chat queue** — type follow-up prompts while pi is still streaming; they're queued and sent in order.
- **one-off scheduled pipes** — schedule a pipe to run at a specific time with `schedule: at <iso>` in the manifest. see [pipes](/pipes).
- **MCP server LAN access** — `screenpipe mcp serve --listen-on-lan --api-key <key>` exposes the MCP HTTP server on your local network with auth required.
- **per-line bbox capture (macOS a11y)** — the timeline now highlights individual lines inside multi-line text nodes, including in Chromium browsers.
- **`on_screen` filter for accessibility search** — narrow `/search` to only what was visible on screen at the time.
- **tinfoil E2EE proxy routes** — AI gateway can now route requests through Tinfoil enclaves end-to-end. see the [privacy filter docs](/privacy-filter).
- **rich onboarding** — onboarding now streams a personalized "here's what i picked up" recap with thumbnails as your first capture rolls in.
- **single recording-quality knob** — one slider replaces the per-monitor quality settings; fixes unreadable text on ultrawides.
- **help: docs + video tutorials** — new cards in **help** link out to written docs and video walkthroughs.
- **in-app file viewer for notifications** — clicking file links in notifications now opens them inside screenpipe instead of the OS file handler.

### updates

- **chat works without sign-in** — the yellow login banner is gone; chat is usable for anyone, with a sign-in nudge only when needed.
- **live OpenAI model catalog** — AI presets fetch the current OpenAI model list instead of being pinned to gpt-4. gpt-5.5 and gpt-5.5-codex are seeded into the ChatGPT preset.
- **API key UX** — regenerate, cancel, and manual-edit flows in **settings → API keys** are now consistent and forgiving.
- **chat reasoning blocks** — the first thought is collapsed by default, queued messages are de-emphasized, and thinking/tool blocks auto-expand when there's no prose.
- **live sidebar indicators** — the sidebar dot pulses while the active chat is streaming, and chat renames broadcast across windows so the sidebar stays in sync.
- **pause/auto-resume on the tray** — live tooltip and a system notification fire when capture pauses or auto-resumes.

### bug fixes

- **stop logging auth tokens and email** in server auth handlers — sensitive values are now redacted.
- **pipes list query** — dropped multi-hundred-MB stdout/stderr payloads; switching pipes is dramatically faster.
- **database** — fixed benign UNIQUE warnings on audio transcription updates and false-positive vision-stall alarms.
- **remote sync** — `screenpipe sync remote` now uses `VACUUM INTO` to take a consistent SQLite snapshot, preventing corrupt copies.
- **chat storage** — silent history loss is fixed; assistant messages persist during streaming (not only after stream-end); concurrent saves no longer collide; and pipe-run sessions on Windows aren't lost to filename sanitization. clipboard fallback when "open URL" fails.
- **new chat in sidebar** — newly created chats now appear immediately instead of after a refresh; "+ new chat" reuses an empty chat.
- **stuck "writing…" indicator** — self-heals when you return to a session.
- **meeting notes** — broken frame images, scrubber ending too early, and Google Calendar–only meetings missing from the list are all fixed; sidebar can be expanded again during a focused meeting.
- **retention** — "delete last N minutes" now actually unlinks `mp4` / `wav` files and drops cached frames.
- **audio** — added a zero-fill watchdog that detects Bluetooth / device-hijack scenarios and only fires once a stream is healthy. fixed `merge-speakers` JSON field names.
- **MCP** — `list-meetings` now parses relative dates ("yesterday", "last monday").
- **AI presets** — ChatGPT OAuth flow polish + stale diagnostics cleared across preset flows. AI gateway promotes `reasoning_content` to content when the main content is empty.
- **owned browser** — survives tray-only mode, falls back to any available window as parent, and fixes cross-origin eval.
- **timeline** — empty-state arrows navigate the right direction, search nav chevrons swapped so ▶ goes forward in time, and tighter per-word search highlights.
- **windows** — overlay timeline Escape key works again when opened from search; restored Windows app build after a `windows-rs` 0.58 import shift.
- **notifications** — clicking the settings link from a notification now navigates correctly; markdown links are no longer double-encoded.
- **tray health** — transient `/health` flaps no longer flash "Error" in the tray.
- **shortcut reminder** — sized correctly when the encrypted store is in use.
- **updater** — errors are no longer propagated to the caller, so a flaky update check doesn't break the app.
- **pipes** — disabling a pipe works even if its connections are missing; tab switching uses client-side navigation instead of anchor jumps; stale-install guard + a UI cancel button on long installs.
- **skill install** — granted the right filesystem scope so installs complete, with proper loading / saved / error states.
- **theme** — dark/light mode inconsistencies with Tauri are resolved.

## week of april 26, 2026

### new features

- **new integrations** — connect [Microsoft 365 and Teams](/connections), Google Docs, Google Sheets, QuickBooks Online, Bitrix24, Loops, Resend, Supabase, and the Pocket AI voice recorder. configure in **settings → connections**.
- **PII privacy filter** — opt-in filter (powered by a Tinfoil enclave) that redacts personal info from `/search` results and chat messages before they leave your machine. toggle from the chat composer. [learn more](/privacy-filter).
- **parallel chats with background streaming** — start multiple chats and switch between them; responses keep streaming in the background. a new chat sidebar shows pinned chats, recents, and pipes scheduled to run.
- **focus-aware capture** — when enabled, screenpipe only records the focused monitor instead of all of them, halving capture cost on multi-display setups. toggle from settings.
- **per-machine pipe favorites** — star pipes you use most. starred pipes sort to the top of the pipe list and get their own filter chip.
- **LAN-access toggle for the API** — bind the local API to `0.0.0.0` so other devices on your network can reach it. API auth is force-enabled when LAN access is on. see the [API docs](/for-developers).
- **`screenpipe sync remote`** — back up your screenpipe data over SSH/SFTP to a server you control. see the [CLI reference](/cli-reference).
- **`screenpipe db {check, recover, cleanup, unlock}`** — new CLI commands to repair database corruption and reclaim disk space. see the [CLI reference](/cli-reference).
- **`screenpipe logout`** — sign out of your screenpipe cloud account from the CLI.
- **connected apps as @mentions in chat** — type `@` in the chat filter popover to scope a question to a specific connected integration.
- **AI quota warnings** — heads-up notice when you're close to your daily limit on weighted models, so you don't get cut off mid-task.
- **browser extension v0.2** — new popup with connection status, an options page, and token-based auth.

### updates

- **CoreAudio Process Tap is now the default for system audio** on macOS 14.4 and later — better quality and no virtual driver required.
- **API keys are now readable and regeneratable** in the UI. user-set custom keys have been removed in favor of a single managed key.
- **Cursor-style inline edit** — click any of your past chat messages to edit and re-run it.
- **redesigned sidebar** — collapse button next to the macOS traffic lights, search opens a focused overlay window with a real keyboard shortcut, and starred pipes outrank running pipes in sort order.
- **timeline calendar** — empty days are now disabled in the day picker and skipped when navigating with arrow keys.
- **chat models hot-swap** — switching a preset's model no longer restarts the chat subprocess.

### bug fixes

- **mic stays connected on sleep/wake** — fixed false-positive disconnects.
- **Microsoft 365 / Teams OAuth** now uses the correct Azure client ID; personal Microsoft accounts are rejected up-front (Teams scope isn't supported there).
- **database reliability** — multibyte string truncation no longer panics, and connection starvation under heavy writes has been resolved.
- **diarization timeouts** — added a `speaker_id` index that fixes 60s+ stalls on long meetings.
- **memories pipeline** silently stopped updating — now back online.
- **macOS memory leaks** — wrapped clipboard capture, monitor enumeration, and focus-tracker callbacks in autorelease pools.
- **search results** — keyword flat mode no longer drops entries that lack text positions.
- **OAuth flows** — auto-refresh of expired tokens in the generic proxy, query params now forwarded through the connection proxy, broader Google Sheets scope, and a fix for tokens being dropped when reconnecting an instance.
- **notifications** — clicking "Open" on a native macOS notification now reliably brings the screenpipe window forward.
- **onboarding** — MCP install uses the bundled bun, so it works without a system bun on `PATH`. WhatsApp connection now resolves bun the same way.
- **calendar permission** — recovers when the macOS Privacy pane reports denied-but-empty.
- **chat polish** — drag-select restored in user messages, "+ new chat" reuses an empty chat instead of spawning duplicates, "Try again" appears when a model returns an empty response, and free models are always allowed regardless of quota.
- **Windows** — restored ARM64 release builds, switched consumer release to SSL.com EV signing, and propagated system root CAs to bundled bun/node via `NODE_EXTRA_CA_CERTS`.
